Note aux correcteurs : il est possible de générer la documentation de nos fonctions avec make doc 

Test_inter
Valide les fonctions d'intersections de rectangle et de verification d'existence d'une intersection

Test_button_2
Affichage d'une image sur un bouton

Test_button_3
Verification randomisé de l'affichage des boutons pour différentes formes/tailles

Test_Placer
Verification de l'absence d'erreurs dynamiques, essais rapides sur le placement en valeurs absolues

Test_Relative
Test rapide sur le placement en valeurs relatives

Test_Geom
Verification complete de la gestion de l'ancrage pour le placer, et rapide de la gestion relative/absolue 

Toplevel
Test général sur les toplevels, leur affichage, redimensionnement et déplacement

Ensiskype
Permet de tester le caractère "scalable" du code, imbrication, parenté entre widget, supperposition

Gridder
Test basique du gridder, vérification de l'absence d'erreur dynamiques

Ensigrid
Test complet pour le gridder inspiré de ensiskype, possibilité de déplacer, redimensionner

Extension
Ensiskype étendu, un niveau d'imbrication supplémentaire et ajout des extensions gridder, tag et radiobutton
